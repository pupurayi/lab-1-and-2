Web Development and Technologies

Lab 1: HTML5 and CSS3 Basics

Objectives:

  • Create a basic HTML5 structure for a webpage.
  • Apply CSS styles to elements to modify their appearance.
  • Understand the concept of CSS selectors.

Tasks:

  1. Create an HTML file with a basic structure (head, body, title,etc.).
  2. Add content to the webpage (headings, paragraphs, images,links).
  3. Apply CSS styles to change the font, colour, background, and layout of the elements.
  4. Experiment with different CSS selectors (e.g., class, id, element).

Lab 2: JavaScript Fundamentals

Objectives:

  • Write simple JavaScript code to interact with HTML elements.
  • Understand basic JavaScript syntax and data types.
  • Create event handlers to respond to user interactions.

Tasks:

  1. Add a JavaScript file to your HTML page.
  2. Write JavaScript code to modify the content of an HTML element.
  3. Create a button that, when clicked, changes the color of aparagraph.
  4. Use JavaScript to validate user input in a form.
